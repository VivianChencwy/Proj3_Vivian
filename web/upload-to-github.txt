GitHub Pages 上传步骤：

1. 首先确保已经生成了数据文件：
   - 运行 notebook 的 Cell 10 和 Cell 11
   - 确保 web/data/tas_2025_by_country.json 存在

2. 在 GitHub 创建新仓库：
   - 访问 https://github.com/new
   - 仓库名：例如 "temperature-map"
   - 选择 Public（公开）
   - 不要勾选 "Initialize this repository with a README"
   - 点击 "Create repository"

3. 在项目文件夹（DSC106\Proj3）打开命令行，运行以下命令：

git init
git add web/
git commit -m "Add temperature map visualization"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main

4. 在 GitHub 启用 Pages：
   - 进入仓库页面
   - 点击 Settings
   - 左侧找到 Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，Folder 选择 "/web"
   - 点击 Save

5. 等待 1-2 分钟后，访问：
   https://你的用户名.github.io/仓库名/
